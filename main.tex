\subsection{Calculations}

The tag, index and offset where computed in the following way:
\begin{lstlisting}
int addr   = Port_Addr.read();
int offset = (addr & 31);
int tag = (addr >> 5);
int index = (tag & 127);     
tag = (tag >> 7); 
\end{lstlisting}
